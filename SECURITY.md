# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in Silo, please report it privately.

Open a [security advisory](https://github.com/divyanshudhruv/silo/security/advisories/new)
or email the maintainer directly.

Do not open a public issue for security vulnerabilities.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1   | No        |
